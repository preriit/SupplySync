# SupplySync Idea Backlog

Use this file to capture ideas without disrupting execution focus.

## Status Definitions
- **parked:** Saved for later, not currently validated.
- **validating:** Gathering evidence (user calls, usage, data).
- **planned:** Selected for near-term implementation.
- **dropped:** Not worth pursuing now.

## Prioritization Heuristic
Prioritize items with highest impact on:
1. Inventory accuracy
2. Stockout reduction
3. Reorder speed

## Idea Template
Copy this block for each idea:

- **Evidence:** <calls, observations, metrics>
